## Setup
### Clone repository
The first thing to do is to clone the repository:

```sh
$ git clone https://github.com/kalickiPawel/eMenu
$ cd eMenu
```

## Tests

To run the tests, `cd` into the directory where `manage.py` is:
```sh
(env)$ py.test
